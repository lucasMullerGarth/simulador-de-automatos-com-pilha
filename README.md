# Simulador de Autômatos com Pilha (PDA)

Este projeto é um **simulador de Autômato com Pilha Não Determinístico (NPDA)** desenvolvido em **Python** com **Tkinter**.  
Ele permite definir estados, alfabeto de entrada, símbolos da pilha, transições e uma cadeia de entrada, exibindo passo a passo a evolução do autômato (estado atual, pilha).

O simulador foi criado para um trabalho da universidade e pode ser útil para estudantes que desejam entender ou visualizar o funcionamento de PDAs.

## Funcionalidades
- Interface gráfica completa em Tkinter  
- Passo a passo da execução do PDA  
- Visualização do estado atual 
- Área de edição para inserir o conjunto de transições

## Como usar
1. Execute o arquivo principal `.py`.  
2. Preencha os campos: Estados, Alfabeto Entrada, Alfabeto Pilha, Estado Inicial, Estados Finais, Símbolo Inicia Pilha e as Transições.
3. Insira a cadeia de entrada.  
4. Clique em **Carregar/Validar**.  
5. Depois clique em **Próximo Passo** para simular.

## Requisitos
- Python
- Tkinter

## Objetivo
O simulador ajuda a compreender melhor o comportamento de autômatos com pilha, especialmente no contexto de disciplinas de teoria da computação ou linguagens formais.
